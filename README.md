# GarupaSimulator
ガルパのカードからイベントごとの最適編成を組むソフトです  
カード情報の取得には[GameOmeGa様のガルパ情報サイト](https://gameo.jp/bang-dream/)をスクレイピングしています  

WPF練習用プロジェクト  
未完成品です
